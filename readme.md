Flanker范式（侧抑制任务）是认知心理学中研究选择性注意与认知控制的经典实验范式，通过设计中心目标刺激与周围干扰刺激，观察被试在忽略干扰并完成目标任务时的反应时和准确率，以揭示注意资源分配和冲突处理机制。以下从定义、实验设计、理论意义、应用领域及变体形式五个方面进行详细说明：

### 定义
通过呈现中心目标刺激（如箭头、字母或情绪面孔）和两侧的干扰刺激（Flankers），考察个体对冲突信息的处理能力。当干扰刺激与目标刺激属性一致时（一致条件），被试反应更快、错误率更低；当两者属性不一致时（不一致条件），反应时延长、错误率增高，这种现象被称为**Flanker效应**。该效应反映了认知控制系统解决冲突的能力，是衡量注意选择性和认知控制水平的重要指标。

### 实验设计

1. **刺激材料**：
   - 本实验以箭头为刺激材料，中央箭头为目标刺激，两侧箭头为干扰刺激。目标箭头可能指向左或右，被试需根据其方向按键反应。
   - 也可以使用字母或图形作为刺激。
2. **实验条件**：
   - **一致条件**：所有干扰刺激与目标刺激方向相同（如中央箭头向左，两侧箭头也向左）。
   - **不一致条件**：所有干扰刺激与目标刺激方向相反（如中央箭头向左，两侧箭头向右）。
   - 也可以包含中性条件。
3. **实验流程**：
   - **输入信息**：包括被试编号、正式实验的blocks数量（nBlocks）
   - **练习阶段**：被试通过1个block(12试次）试次熟悉任务要求，设置准确率阈值（如75%）作为进入正式实验的标准。
   - **正式实验**：在n个block(n*20试次）中，随机呈现不同条件的试次，记录被试的反应时和正确率。每个条件通常需至少30次重复以确保数据稳定性。
   - **数据记录**：包括试次编号、刺激条件、反应时、反应正确性等变量。分析前需剔除错误试次和极端反应时（如±3个标准差之外的数据）。

### 理论意义

1. **揭示注意的选择性机制**：Flanker效应表明，干扰刺激会竞争注意资源，导致不一致条件下的反应时延长。这反映了注意对目标刺激的选择性聚焦和对干扰刺激的抑制过程。
2. **阐明认知控制中的冲突监测与解决**：当干扰刺激与目标刺激不一致时，会产生两种相互冲突的反应倾向，需要额外的认知控制资源来解决冲突。这一过程涉及前扣带回（ACC）对冲突信号的监测和背侧前额叶（DLPFC）对控制水平的调节。
3. **支持自动加工与控制加工的相互作用理论**：Flanker效应反映了自动加工（对干扰刺激的无意识反应）与控制加工（对目标刺激的有意识选择）之间的动态平衡。不一致条件下的反应时延长，表明控制加工需要克服自动加工的干扰。

### 应用领域

1. **注意研究**：通过比较不同条件下被试的反应时和错误率，考察注意资源的分配情况和干扰刺激对注意加工过程的影响机制。
2. **认知控制研究**：Flanker范式常用于研究执行功能、认知控制和冲突监控等高级认知功能。例如，考察前一个试次的冲突水平对当前试次表现的影响（冲突适应效应）。
3. **情绪调节研究**：使用情绪面孔作为刺激材料的情感版Flanker范式，可考察情绪干扰对注意和认知控制的影响。
4. **特殊群体研究**：Flanker范式可用于评估儿童、老年人或临床患者（如注意缺陷多动症患者）的冲突控制能力。研究发现，注意缺陷多动症患者的冲突控制能力相对于正常组通常受到显著削弱。
